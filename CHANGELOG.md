# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-05-18

- Initial public release.
- Mechanical arms can be armed/disarmed with melee and ranged weapons.
- Ranged aiming, ammo loading from arm inputs, and RPM-scaled timing.
- Melee combat now uses a fake-player attack path for better weapon compatibility.
- Item capability exposure for held weapon and loaded ammo state (for HUD mods like Jade).
